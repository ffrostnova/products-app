# Products App

Тестовое задание - SPA приложение для работы с продуктами.

## Стек

- Next.js 14
- TypeScript
- Zustand
- Tailwind CSS
- Axios

## Функционал

- Список продуктов с карточками
- Лайки и удаление
- Фильтрация (все/избранное)
- Страница детального просмотра
- Создание нового продукта
- Редактирование продукта
- Поиск и фильтрация по категориям
- Пагинация

## Установка

```bash
npm install
npm run dev
```

## Деплой

Проект настроен для деплоя на GitHub Pages через GitHub Actions.
