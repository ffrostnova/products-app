# Products App

SPA приложение для управления продуктами с использованием Next.js, TypeScript и Zustand.

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Zustand** - управление состоянием
- **Tailwind CSS** - стилизация
- **Axios** - HTTP запросы

## Функциональность

### Основные задачи:
- ✅ Список продуктов с карточками
- ✅ Лайки и удаление продуктов
- ✅ Фильтрация (все/избранное)
- ✅ Страница детального просмотра продукта
- ✅ Создание нового продукта с валидацией

### Бонусы:
- ✅ Пагинация списка
- ✅ Редактирование продукта
- ✅ Дополнительная фильтрация
- ✅ Поиск по продуктам

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Экспорт статических файлов
npm run export
```

## Структура проекта

```
products-app/
├── app/                    # Next.js App Router
│   ├── products/          # Страницы продуктов
│   ├── create-product/    # Создание продукта
│   └── layout.tsx        # Основной layout
├── components/            # React компоненты
├── store/                # Zustand store
├── types/                # TypeScript типы
└── utils/                # Утилиты
```

## API

Используется [Fake Store API](https://fakestoreapi.com/) для получения данных о продуктах.

## Деплой на GitHub Pages

### Настройка репозитория

1. Создайте репозиторий на GitHub
2. Добавьте remote:
   ```bash
   git remote add origin https://github.com/ваш-username/ваш-репозиторий.git
   ```

### Настройка GitHub Pages

1. Перейдите в Settings → Pages
2. В разделе "Source" выберите "GitHub Actions"
3. При каждом push в main ветку будет автоматически собираться и деплоиться проект

### Ручной деплой

```bash
# Установка зависимостей
npm install

# Сборка проекта
npm run build

# Файлы будут в папке /out
# Загрузите содержимое папки /out на GitHub Pages
```

### Важно для GitHub Pages

- В `next.config.js` уже настроен `output: 'export'` для статической генерации
- Используются относительные пути для роутинга
- Изображения настроены как `unoptimized: true` для статического экспорта
